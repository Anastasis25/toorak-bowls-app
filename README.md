# Toorak Bowling Club - Member Portal

A Progressive Web App (PWA) for Toorak Bowling Club members.

**Est. 1913 • 9 Mandeville Cres, Toorak VIC 3142**

## Features

### ✅ Availability Management
- Mark yourself available/unavailable for upcoming matches
- Set position preferences (Skip, Third, Second, Lead)
- Separate tracking for pennant and social bowls

### 📅 Rink Booking System
- Book practice rinks with available time slots
- View your current bookings
- Cancel bookings easily

### 🎉 Social Events
- View upcoming club events (BBQs, Friday night bowls, barefoot bowls)
- RSVP with one tap
- See who else is attending

### 🏛️ Function Room Hire
- Quick enquiry form for room bookings
- Main hall (up to 100 people), bar, and outdoor green
- Perfect for birthdays, corporate events, private functions

### 🔗 BowlsLink Integration
Designed to complement (not replace) Bowls Link:
- Use this for availability, bookings, and social coordination
- Official draws and results stay in Bowls Link

## Installation

### For Members:
1. Visit the app URL on your phone
2. Tap "Add to Home Screen" when prompted
3. The app icon will appear on your home screen

### For Club Admins:
This is a prototype. To make it production-ready:
1. Add a backend API for data sync
2. Integrate with your club's database
3. Add authentication for members
4. Connect to weather API for live data
5. Optional: Integrate with Bowls Link API

## Deployment to GitHub Pages

1. Create a new repository called `toorak-bowls-app`
2. Upload `toorak-bowls-v2.html` and rename it to `index.html`
3. Go to Settings → Pages
4. Select "Deploy from main branch"
5. Your app will be live at: `https://[your-username].github.io/toorak-bowls-app/`
